What is SentrySearch?

SentrySearch is a natural-language search tool for dashcam and security footage. It turns a question into a relevant clip so you can jump straight to the moment that matters.

Common use cases

  • Find the moment a car cut in front of me
  • Find when someone approached the driveway
  • Find the clip with a red truck near the stop sign

How it works

SentrySearch indexes video moments, matches the language in your request, and surfaces the clip that best fits the event you described.

FAQ

Does SentrySearch upload my videos?
No. You can keep the workflow local and control where your footage lives.
Can I run SentrySearch locally?
Yes. It is designed to run on your own machine for local searches.
Is this a Tesla Sentry Mode tool only?
No. SentrySearch is built for dashcam and security footage, not just Tesla Sentry Mode clips.
